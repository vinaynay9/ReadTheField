READ THE FIELD

High-Level Overview

Read the Field is a Monte Carlo NFL player simulation engine built on nflreadr. It focuses on uncertainty, distributions, and variance — not single-point predictions.

Why NFL Outcomes Are Random

Game script, matchups, coaching decisions, and randomness all shape player outcomes. The signal is real, but the noise is strong — so a distribution is the only honest answer.

Models Used

  • Monte Carlo simulation
  • Linear regression baseline
  • Neural networks (future)
