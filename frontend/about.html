READ THE FIELD

About Read the Field

Read the Field is a probabilistic NFL player projection system that focuses on offensive skill positions and kickers. Instead of emitting a single point estimate, it produces distributional outcomes so you can see ranges, percentiles, and risk.

Today the system supports matchup-aware simulations for historical replays, upcoming games, and hypothetical scenarios. Every run is grounded in leakage-safe features built from data available before kickoff.

What You Can Do

  • Explore player outcome distributions for specific matchups.
  • Compare historical matchups vs. counterfactual opponents.
  • See uncertainty bands instead of a single prediction.

Future Iterations

  • Fantasy lineup inputs with full-week distributions.
  • True odds vs. betting lines and market comparisons.
  • Expanded feature engineering and richer context layers.
  • Model comparisons (neural nets vs. Monte Carlo vs. linear models).
